## 类: TouchBarLabel
>在触摸条中为本机macOS应用程序创建一个label标签

进程: [主进程](../tutorial/quick-start.md#main-process)       

### `new TouchBarLabel(options)` _实验功能_
>用途:**创建一个新的label标签**

* `options` Object
  * `label` String (可选) - 要显示的文本
  * `textColor` String (可选) - 十六进制的文本颜色, 如`#ABCDEF`.


### 实例属性
 `TouchBarLabel`有以下属性:

#### `touchBarLabel.label`
>属性:**label标签的当前文本**

#### `touchBarLabel.textColor`
>属性:**label标签的当前文本颜色**
